Proposito del Sitio

El propósito de este sitio Web consiste en dar inicio a la construcción de un proyecto haciendo uso del lenguaje de marcado estándar para la construcción de páginas Web, por medio de editores de código, estructuración de código y el uso de etiquetas. De tal manera que cada uno de Los elementos HTML sea interpretado por el navegador y permitan mostrar el contenido tal cual se espera reflejar.

Las Etiquetas Basicas de HTML Atributos y Valores

Definicion de HTML

HTML se define como el lenguaje de marcado de hipertexto utilizado para el desarrollo de páginas web, es un lenguaje compuesto de una serie de etiquetas o marcas que permiten definir el contenido y asi mismo la apariencia de las páginas web que conocemos hoy en dia. HTML al ser un lenguaje de Texto permite insertar elementos tales como imágenes, videos, listas. Además de ordenar y etiquetar diversos documentos dentro de una lista, este lenguaje es el que se utiliza para especificar los nombres de las etiquetas que se utilizaran dándole una estructura y un orden, se pude decir que se un lenguaje muy simple y general que sirve para definir otros lenguajes que tiene que ver con el formato y estilos de los documentos. En el Lenguaje html el texto se crea a partir de etiquetas que permiten interconectar diversos conceptos y formatos.

Atributos y Valores HTML

Un atributo se describe como un conjunto de palabras especiales utilizadas al interior de la etiqueta de apertura - de tal manera que permita controlar el comportamiento de un determinado elemento, Los atributos permiten personalizar las etiquetas html.

  • Ejemplo de Atributo

    • En la etiqita "a" que se refiere a la etiqueta de enlace que nos direcciona otra pagina por medio del atributo "href"

    En la Etiqueta "Img" que hace referencia a Imagenes se adiciona el atributo "src" indicando la ruta donde se encuentran almacenedas las Imagenes ademas podemos encontrar "alt" que seria texto alternanivo de la imagen en dado caso que no se encuntre la Imagen nos muestre el texto

Tipos de Atributos

Aunque cada una de las etiquetas HTML define sus propios atributos, encontramos algunas comunes a muchas o casi todas las etiquetas, que se dividen en cuatro grupos segun su funcionalidad.

  • Atributo Basicos:
    Los atributos bàsicos se utilizan en la mayoria de etiquetas html y xhtml aunque adquieren mayor sentido cuando se utilizan hojas de estilos en cascada.
  • Atributos de Internacionalización:
    Estos atributos se utilizan en aquellas paginas que muestran sus contenidos en varios Idiomas y las que quieren indicar de forma explicita el idioma de sus contenidos.
  • Atributo de Eventos
    Estos atributos se utilizan en las páginas web que incluyen código JavaScript para realizar acciones dinámicas sobre los elementos de la página.
  • Atributos de Foco:
    Se le denomina foco o focus, cuando un control o elemento del documento ha sido seleccionado. Cuando ese elemento deja de estar seleccionado, pierde el foco y es el nuevo elemento seleccionado el que se dice que tiene el foco.
